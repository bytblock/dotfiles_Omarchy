# Deploy to Linode

Connect to the Linode server and deploy the current project or run the specified command.

Server: root@$LINODE_IP (Ubuntu 22.04)

## Default behavior
1. SSH into Linode server
2. Pull latest code from git (if applicable)
3. Restart services as needed
4. Report status

## If argument provided
Run the specified command on the Linode server.

$ARGUMENTS
