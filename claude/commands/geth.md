# Geth Node Status

Check the status of the geth (Go Ethereum) node on the Linode server.

## Tasks
1. SSH to Linode and check if geth is running
2. Check sync status (blocks synced, peers connected)
3. Check disk usage for blockchain data
4. Report any errors from geth logs (last 20 lines)

If geth is not installed yet, offer to install and configure it.
